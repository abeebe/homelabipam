# Homelab IPAM

A modern, self-hosted IP Address Management (IPAM) tool built for homelabs.

## Features

- Network & VLAN management
- Device tracking
- UniFi sync (planned)
- Proxmox sync (planned)
- Docker-first deployment

## Tech Stack

- React + TypeScript
- Node + Express
- Prisma ORM
- PostgreSQL
